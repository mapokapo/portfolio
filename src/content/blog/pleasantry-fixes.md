title: "Pleasantry fixes"
published: 2022-10-27
description:
  "Mobile layout tweaks, horizontal showcase tiles, working blog post pages,
  Firefox fixes, and expandable intro slides - small changes that add up."
---

I've made several changes to improve the **mobile experience**. The layout now
has reduced whitespace and smaller text sizes, so that only one main element is
shown at a time, and it actually fits fully on the screen.

The **showcase tiles** have been reworked into a horizontal list on mobile, and
I also added direct _links to GitHub projects_ as suggested.

The biggest news: **blog posts finally work!** You can now click on any post
from the main page and it will open into a full content view. It's still a bit
barebones, but it's a proper start.

I also fixed a bunch of issues in **Firefox** and made the site generally more
responsive and easier to read. That includes switching to the horizontal list on
mobile and making the **introductory slides expandable** when viewed on smaller
screens.
